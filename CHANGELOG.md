# Changelog

All notable changes to BegireX are documented here.

## [0.1.0] - 2026-07-15

### Added

- Initial public release of the Tauri desktop download manager.
- Local queue controls, download-format inspection, preset management, and
  SQLite-backed persistence.
- Onboarding that detects or downloads yt-dlp and FFmpeg.
- Light and bundled build flavors.
